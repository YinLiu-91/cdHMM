Detailed Description

template<typename obs_type>
class cdHMM::laplaceHMM< obs_type >

HMM with laplace emission
prob( Emit O | state i ) = (1 / 2b ) exp( -|x - mu|/b )
mu_i = B[i][0]
b_i = B[i][1]
